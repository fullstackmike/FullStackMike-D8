INTRODUCTION
------------

This is a portfolio theme which is based upon bootstrap 4 Framework. Any engineer who wants to make his/her portfolio website can try this theme once.

 REQUIREMENTS
 ------------ 

Bootstrap (https://www.drupal.org/project/bootstrap_barrio )

 INSTALLATION
 ------------

1. Download Bootstrap (https://www.drupal.org/project/bootstrap_barrio) and locate this in /theme folder

2. Download portfolio (https://www.drupal.org/project/portfolio_theme) and locate this in /theme folder as sub-theme

CONFIGURATION
-------------

1. go to /admin/appearance/settings/ and enable portfolio  theme.

2. Block settings:
     go to /admin/structure/block  and remove extra block which is not required
